patchit¶

tollan.utils.misc.patchit(obj, name, pass_patched=False)[source]¶

Return a decorator that replace attribute with decorated item.

Parameters
obj: object

The object to alter.

name: str

The name of the attribute to replace.

pass_patched: bool

If set to True, the original attribute will be passed to as the first argument to the decorated function.
