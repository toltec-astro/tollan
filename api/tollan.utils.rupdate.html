rupdate¶

tollan.utils.rupdate(d, u)[source]¶

Update dict recursively.

This will update d with items in u in a recursive fashion. Keys in d will not be deleted if they are found in u.

Parameters
d, udict
Returns
None

Dict d is updated in place.

Notes
